PostgreSQL

Instalar PostgreSQL

# apt update
# apt install postgresql

Entrar en el gestor de base de datos

# su - postgres
$ psql

Crear el usuario y la base de datos con la que vamos a trabajar

postgres=# CREATE USER usuario PASSWORD 'clave';
postgres=# CREATE DATABASE basededatos WITH OWNER usuario;
postgres=# \q

Salir hasta nuestro prompt

$ ^D
# ^D

Ya podemos trabajar con nuestro usuario y nuestra base de datos

$ psql -h 127.0.0.1 -U usuario basededatos
