Android Studio

>>> Descargar versión Linux(64-bit) desde https://developer.android.com/studio#downloads

>>> Desde la línea de comandos ir al directorio donde se quiera instalar android-studio
$ cd <directorio donde se quiera trabajar>

>>> copiar el fichero descargado al directorio actual

>>> Descomprimir y borrar el fichero

$ tar vxfz android-studio-XXXX.X.X.XX-linux.tar.gz
$ rm android-studio-XXXX.X.X.XX-linux.tar.gz
$ cd <directorio-generado>

>>> Ejecutar Android Studio
$ ./bin/studio.sh
