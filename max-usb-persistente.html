Crear un USB persistente de MAX

MAX

Instalación

Instalar mkusb:

sudo add-apt-repository ppa:mkusb/ppa
sudo apt update
sudo apt install --install-recomends mkusb mkusb-nox usb-pack-efi

Ejecutar mkusb:

mkusb

Seleccionar versión "dbus"
Seleccionar "Install (make a boot device)"
Seleccionar la ISO que se quiere grabar previamente descargada
Seleccionar el dispositivo USB donde se quiere guardar ¡cuidado!
Seleccionar "'Persistent live' - only Debian and Ubuntu"
Seleccionar el tamaño de persistencia (al máximo posible)
Seleccionar "Go"

Al finalizar extraer el USB

Ejecución

Arrancar el PC con el USB conectado
Seleccionar arranque con USB (generalmente F12 al arrancar)

Configuración

Al ser una unidad Live algunos elementos del escritorio y de la configuración siempre se reinician y nos gustaría mantenerlas en los sucesivos arranques. Es cuestión de gustos pero yo prefiero que no aparezcan los iconos de Instalar en el disco duro y otros. Para ello realizaremos varios scripts que arrancaremos con crontab o como servicios en /etc/init.d.

NOTA

Ninguna herramienta Windows, ni tan siquiera Rufus, consigue la persistencia que nosotros pretendemos.
